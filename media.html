How to add screenshots:
1. Upload your image to Google Drive → right-click → "Get link" → set to "Anyone with the link"
2. Copy the file ID from the share URL: https://drive.google.com/file/d/FILE_ID/view
3. Use this as src: https://drive.google.com/uc?export=view&id=FILE_ID
4. Paste a new photo-item block below and fill in the src and caption.

How to add YouTube clips:
1. Go to the YouTube video → click Share → Embed
2. Copy the embed URL (looks like: https://www.youtube.com/embed/VIDEO_ID)
3. Paste a new yt-card block below and fill in the src, title, and sub-label.
